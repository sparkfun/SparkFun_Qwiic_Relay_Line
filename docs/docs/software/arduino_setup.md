# SparkFun I<sup>2</sup>C Expander Arduino Library

The SparkFun I2C Expander Library makes working with 4-bit and 8-bit I2C digital I/O expanders very easy.. You can install the library with the Arduino [Library manager](https://docs.arduino.cc/software/ide-v2/tutorials/ide-v2-installing-a-library/) tool by searching for "SparkFun I2C Expander". If you'd prefer a manual install, you can download a ZIP of the library [here](https://github.com/sparkfun/SparkFun_I2C_Expander_Arduino_Library/archive/refs/heads/main.zip).
